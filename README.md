# A really basic 11ty starter.

## What's been setup/installed:

- 11ty
- .eleventy.js
  ```
  return {
    markdownTemplateEngine: "njk",
    dataTemplateEngine: "njk",
    htmlTemplateEngine: "njk",
    dir: {
      input: "src",
      output: "dist",
    },
  };
  ```
- .gitignore
- package.json
- index.md

## Running locally

```
npm start
```
